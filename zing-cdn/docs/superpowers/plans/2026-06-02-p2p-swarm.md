# P2P Swarm Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Wire a real libp2p P2P swarm into zing-cdn-core, replacing the ZingP2pNode stub, enabling L1 blob resolution via Kademlia DHT + blob data transfer over libp2p request-response.

**Architecture:** ZingBehaviour composite (Kademlia + custom request-response + Identify + Ping) runs inside a Swarm owned by a background tokio task. The Resolver communicates via mpsc command channel. Blob data is transferred inline in the response for MVP (streaming chunks in a future iteration).

**Tech Stack:** libp2p 0.54 (kad, request-response, identify, ping, quic, noise, yamux), serde_json, serde

### Task 1: Implement Protocol Codec + Messages

**Files:**
- Modify: `zing-cdn-core/src/p2p/protocol.rs`

Rewrite the file with protocol message types and a custom request-response codec using serde_json.

```rust
use serde::{Deserialize, Serialize};
use std::io;
use async_trait::async_trait;
use futures::prelude::*;
use serde_json;

pub const ZING_CDN_BLOB_PROTOCOL: &str = "/zing-cdn/data/1.0";

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct BlobRequest {
    pub blob_id: [u8; 32],
    pub version: u8,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct BlobResponse {
    pub size: Option<u64>,
    #[serde(with = "serde_bytes")]
    pub data: Option<Vec<u8>>,
}

impl BlobResponse {
    pub fn have(data: Vec<u8>) -> Self {
        Self { size: Some(data.len() as u64), data: Some(data) }
    }

    pub fn not_found() -> Self {
        Self { size: None, data: None }
    }

    pub fn is_have(&self) -> bool {
        self.data.is_some()
    }
}

#[derive(Debug, Clone)]
pub struct JsonProtocolCodec;

#[async_trait]
impl libp2p::request_response::ProtocolCodec for JsonProtocolCodec {
    type Request = BlobRequest;
    type Response = BlobResponse;

    async fn read_request<T>(&mut self, io: &mut T) -> io::Result<Self::Request>
    where
        T: AsyncRead + AsyncWrite + Unpin + Send,
    {
        let mut len_buf = [0u8; 4];
        io.read_exact(&mut len_buf).await?;
        let len = u32::from_be_bytes(len_buf) as usize;
        let mut buf = vec![0u8; len];
        io.read_exact(&mut buf).await?;
        serde_json::from_slice(&buf).map_err(|e| io::Error::new(io::ErrorKind::InvalidData, e))
    }

    async fn read_response<T>(&mut self, io: &mut T) -> io::Result<Self::Response>
    where
        T: AsyncRead + AsyncWrite + Unpin + Send,
    {
        let mut len_buf = [0u8; 4];
        io.read_exact(&mut len_buf).await?;
        let len = u32::from_be_bytes(len_buf) as usize;
        let mut buf = vec![0u8; len];
        io.read_exact(&mut buf).await?;
        serde_json::from_slice(&buf).map_err(|e| io::Error::new(io::ErrorKind::InvalidData, e))
    }

    async fn write_request<T>(&mut self, io: &mut T, req: Self::Request) -> io::Result<()>
    where
        T: AsyncRead + AsyncWrite + Unpin + Send,
    {
        let buf = serde_json::to_vec(&req).map_err(|e| io::Error::new(io::ErrorKind::InvalidData, e))?;
        let len = (buf.len() as u32).to_be_bytes();
        io.write_all(&len).await?;
        io.write_all(&buf).await?;
        Ok(())
    }

    async fn write_response<T>(&mut self, io: &mut T, res: Self::Response) -> io::Result<()>
    where
        T: AsyncRead + AsyncWrite + Unpin + Send,
    {
        let buf = serde_json::to_vec(&res).map_err(|e| io::Error::new(io::ErrorKind::InvalidData, e))?;
        let len = (buf.len() as u32).to_be_bytes();
        io.write_all(&len).await?;
        io.write_all(&buf).await?;
        Ok(())
    }
}
```

### Task 2: Implement Data Handler (BlobStore queries)

**Files:**
- Rewrite: `zing-cdn-core/src/p2p/handler.rs`

The handler bridges the incoming `BlobRequest` with the local `BlobStore`. It's called by the behaviour when an inbound request arrives.

```rust
use crate::cache::store::BlobStore;
use crate::p2p::protocol::BlobRequest;
use crate::p2p::protocol::BlobResponse;
use std::sync::Arc;
use tokio::sync::RwLock;

pub type BlobStoreHandle = Arc<RwLock<BlobStore>>;

pub async fn handle_inbound_request(
    store: &BlobStoreHandle,
    request: BlobRequest,
) -> BlobResponse {
    let blob_id_hex = hex::encode(request.blob_id);
    tracing::info!(blob_id = %blob_id_hex, "handling inbound blob request");

    let store = store.read().await;
    match store.get(&blob_id_hex) {
        Ok(Some(data)) => {
            tracing::info!(blob_id = %blob_id_hex, size = data.len(), "responding HAVE");
            BlobResponse::have(data)
        }
        Ok(None) => {
            tracing::info!(blob_id = %blob_id_hex, "responding NOT_FOUND");
            BlobResponse::not_found()
        }
        Err(e) => {
            tracing::error!(blob_id = %blob_id_hex, error = %e, "store read error");
            BlobResponse::not_found()
        }
    }
}
```

```rust
use crate::p2p::behaviour::ZingBehaviour;

use std::sync::Arc;

pub const ZING_CDN_BLOB_PROTOCOL: &str = "/zing-cdn/data/1.0";

#[derive(Debug)]
pub enum P2pCommand {
    AnnounceBlob { blob_id: [u8; 32] },
    FindProviders {
        blob_id: [u8; 32],
        reply: oneshot::Sender<Vec<PeerId>>,
    },
    FetchBlob {
        peer_id: PeerId,
        blob_id: [u8; 32],
        reply: oneshot::Sender<ZingResult<Vec<u8>>>,
    },
    AddBootstrapPeer {

        addr: Multiaddr,
    },

    GetConnectedPeers {

pub struct ZingP2pNode {
    local_peer_id: PeerId,
    command_tx: mpsc::Sender<P2pCommand>,
    store: BlobStoreHandle,
}

impl ZingP2pNode {
    pub fn new(store: BlobStoreHandle) -> (Self, mpsc::Receiver<P2pCommand>) {
        let local_key = identity::Keypair::generate_ed25519();
        let local_peer_id = local_key.public().to_peer_id();
        let (command_tx, command_rx) = mpsc::channel::<P2pCommand>(64);

        let node = Self { local_peer_id, command_tx, store };

        (node, command_rx)
    }

    pub fn local_peer_id(&self) -> PeerId {
        self.local_peer_id
    }

    pub fn command_tx(&self) -> mpsc::Sender<P2pCommand> {
        self.command_tx.clone()
    }

    pub async fn run(
        command_rx: mpsc::Receiver<P2pCommand>,
        store: BlobStoreHandle,
        listen_on: Multiaddr,
        bootstrap_addrs: Vec<(PeerId, Multiaddr)>,
    ) -> ZingResult<()> {
        Self::run_impl(command_rx, store, listen_on, bootstrap_addrs).await
    }
}

// The actual run implementation is in a private module to keep the public API clean.
// We use `run_impl` which builds the swarm and runs the event loop.
impl ZingP2pNode {
    async fn run_impl(
        mut command_rx: mpsc::Receiver<P2pCommand>,
        store: BlobStoreHandle,
        listen_addr: Multiaddr,
        bootstrap_addrs: Vec<(PeerId, Multiaddr)>,
    ) -> ZingResult<()> {
        let local_key = identity::Keypair::generate_ed25519();
        let local_peer_id = local_key.public().to_peer_id();

        let mut swarm = SwarmBuilder::new(local_key, ZingBehaviour::new)
            .with_quic();
            // TODO: fill in complete SwarmBuilder chain
            // returns Swarm<ZingBehaviour>

        swarm.listen_on(listen_addr)
            .map_err(|e| ZingError::P2PNetwork(e.to_string()))?;

        // Add bootstrap peers
        for (peer_id, addr) in &bootstrap_addrs {
            swarm.behaviour_mut().kad.add_address(peer_id, addr.clone());
        }

        // Bootstrap DHT
        if let Err(e) = swarm.behaviour_mut().kad.bootstrap() {
            tracing::warn!(error = %e, "kad bootstrap failed");
        }

        loop {
            tokio::select! {
                Some(cmd) = command_rx.recv() => {
                    Self::handle_command(&mut swarm, cmd)?;
                }
                event = swarm.next() => {
                    if let Some(event) = event {
                        Self::handle_swarm_event(&mut swarm, event, &store);
                    }
                }
            }
        }
    }

            P2pCommand::FindProviders { blob_id, reply } => {
                let key = libp2p::kad::RecordKey::new(&blob_id);
                swarm.behaviour_mut().kad.inner.get_providers(key);
                // The reply is sent asynchronously when the DHT query completes.
                // For MVP, we store the reply sender in a pending map.
                // This requires adding a field to ZingP2pNode for pending queries.
                todo!();
            }
            P2pCommand::FetchBlob { peer_id, blob_id, reply } => {
                let request = BlobRequest { blob_id, version: 1 };
                swarm.behaviour_mut().data.send_request(&peer_id, request);
                // The reply is sent asynchronously when the response arrives.
                todo!();
            }
            P2pCommand::AddBootstrapPeer { peer_id, addr } => {
                swarm.behaviour_mut().kad.inner.add_address(&peer_id, addr);
            }
            P2pCommand::Bootstrap => {
                swarm.behaviour_mut().kad.inner.bootstrap()
                    .map_err(|e| ZingError::P2PNetwork(e.to_string()))?;
            }
            P2pCommand::GetConnectedPeers { reply } => {

            }
        }

    }
```

### Task 5: Complete the Event Loop (handle pending queries + responses)

This task fills in the `todo!()` placeholders from Task 4. The node needs to track pending `FindProviders` and `FetchBlob` queries by storing oneshot senders indexed by query ID.

The approach: For each outbound request, we generate a unique `RequestId` and store the oneshot sender in a `HashMap`. When the response arrives via the `SwarmEvent`, we look up the sender and deliver the result.

**Files:**
- Modify: `zing-cdn-core/src/p2p/node.rs`

```rust

use libp2p::request_response::{self, RequestId};
use libp2p::kad::{self, QueryId};

struct PendingFindProviders {
    reply: oneshot::Sender<Vec<PeerId>>,
}

struct PendingFetch {
    reply: oneshot::Sender<ZingResult<Vec<u8>>>,
}

// These are stored in the run_impl function's scope:
let mut pending_finds: HashMap<QueryId, PendingFindProviders> = HashMap::new();
let mut pending_fetches: HashMap<RequestId, PendingFetch> = HashMap::new();
```

When handling the response:

```rust
        request_response::Message::Response { request_id, response } => {

                    }
                    _ => {
                        let _ = pending.reply.send(Err(ZingError::BlobNotFound(hex::encode(blob_id))));
```

### Task 6: Wire Resolver L1 Resolution

**Files:**
- Modify: `zing-cdn-core/src/mesh/resolver.rs`

Add P2pNode integration to `Resolver`:

```rust
use tokio::sync::mpsc;
use crate::p2p::node::{P2pCommand, ZingP2pNode};

pub struct Resolver {
    // ... existing fields ...
    p2p_command_tx: Option<mpsc::Sender<P2pCommand>>,
}

impl Resolver {
    pub fn new(
        store: Arc<RwLock<BlobStore>>,
        pinning: Arc<RwLock<PinningManager>>,
        eviction: Arc<RwLock<EvictionManager>>,
        walrus_client: Arc<WalrusL3Client>,
        verifier: Arc<BlobVerifier>,
        reputation: Arc<RwLock<PeerReputationTable>>,
    ) -> Self {
        Self {
            store,
            pinning,
            eviction,
            walrus_client,
            verifier,
            reputation,
            p2p_command_tx: Default::default(),
        }
    }

    pub fn set_p2p_channel(&mut self, tx: mpsc::Sender<P2pCommand>) {
        self.p2p_command_tx = Some(tx);
    }

    pub async fn resolve(&self, blob_id: &BlobId) -> ZingResult<ResolveResult> {
        let blob_id_hex = blob_id.to_string();
        tracing::info!(blob_id = %blob_id_hex, "resolving blob request");

        // Layer 0: Local cache
        {
            let store = self.store.read().await;
            if let Some(data) = store.get(&blob_id_hex)? {
                tracing::info!(blob_id = %blob_id_hex, "L0: blob found in local cache");
                return Ok(ResolveResult {
                    data,
                    resolution: BlobResolution::LocalCache,
                    cached: true,
                });
            }
        }

        // Layer 1: P2P peers
        if let Some(ref tx) = self.p2p_command_tx {
            if let Ok(data) = self.resolve_from_l1(blob_id, &blob_id_hex, tx).await {
                return Ok(data);
            }
        }

        // Layer 3: Walrus
        self.resolve_from_walrus(blob_id, &blob_id_hex).await
    }

    async fn resolve_from_l1(
        &self,
        blob_id: &BlobId,
        blob_id_hex: &str,
        tx: &mpsc::Sender<P2pCommand>,
    ) -> ZingResult<ResolveResult> {
        tracing::info!(blob_id = %blob_id_hex, "L1: looking up DHT providers");

        let (find_reply, find_rx) = tokio::sync::oneshot::channel();
        tx.send(P2pCommand::FindProviders {
            blob_id: *blob_id.as_ref(),
            reply: find_reply,
        }).await.map_err(|_| ZingError::P2PNetwork("p2p channel closed".into()))?;

        let peers = tokio::time::timeout(Duration::from_secs(5), find_rx)
            .await
            .unwrap_or(Ok(vec![]))
            .unwrap_or(vec![]);

        if peers.is_empty() {
            tracing::info!(blob_id = %blob_id_hex, "L1: no providers found in DHT");
            return Err(ZingError::NoPeersAvailable(blob_id_hex.to_string()));
        }

        // Pick highest-reputation peer
        let reputation = self.reputation.read().await;
        let peer = peers.into_iter()
            .max_by_key(|p| {
                let key = p.to_string();
                reputation.peer_score(&key).unwrap_or(0)
            })
            .expect("at least one peer");

        tracing::info!(blob_id = %blob_id_hex, peer = %peer, "L1: fetching from peer");

        let (fetch_reply, fetch_rx) = tokio::sync::oneshot::channel();
        tx.send(P2pCommand::FetchBlob {
            peer_id: peer,
            blob_id: *blob_id.as_ref(),
            reply: fetch_reply,
        }).await.map_err(|_| ZingError::P2PNetwork("p2p channel closed".into()))?;

        let data = tokio::time::timeout(Duration::from_secs(30), fetch_rx)
            .await
            .map_err(|_| ZingError::P2PNetwork("fetch timeout".into()))?
            .map_err(|_| ZingError::P2PNetwork("fetch failed".into()))??;

        // Verify blob
        {
            let metadata = self.walrus_client.fetch_metadata(blob_id).await?;
            self.verifier.verify_blob_against_metadata(&metadata, &data)?;
        }

        // Cache the blob
        {
            let store = self.store.write().await;
            store.put(blob_id_hex, &data)?;
        }
        // Run eviction
        {
            let pinning = self.pinning.read().await;
            self.eviction.write().await.run(&pinning)?;
        }

        // Record success
        self.reputation.write().await.record_success(&peer.to_string());

        tracing::info!(blob_id = %blob_id_hex, peer = %peer, "L1: blob fetched and verified");
        Ok(ResolveResult {
            data,
            resolution: BlobResolution::L1Peer,
            cached: false,
        })
    }
}
```
